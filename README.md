## How to deploy (5 minutes, free forever)

1. Create a free account at **github.com**
2. Click **+** → **New repository** → name it `exampath2027` → **Public** → Create
3. Click **Add file → Upload files**
4. Upload ALL files from this zip (index.html, README.md, _config.yml, .nojekyll, 404.html)
5. Click **Commit changes**
6. Go to **Settings → Pages → Source → Deploy from branch → main → / (root)** → Save
7. Wait 2 minutes → Your link is live at: `https://YOUR-USERNAME.github.io/exampath2027`

## Activate Google Analytics (when ready)

Google Analytics is already built into the file — just needs your ID activated.

### Step 1 — Create Google Analytics account
1. Go to **analytics.google.com**
2. Sign in with your Google account
3. Click **Start measuring**
4. Account name: `ExamPath 2027` → Next
5. Property name: `ExamPath 2027` → Next  
6. Select **Web** as platform
7. Enter your GitHub Pages URL: `https://YOUR-USERNAME.github.io/exampath2027`
8. Click **Create stream**
9. Copy your **Measurement ID** — it looks like `G-ABC123XYZ`

### Step 2 — Add your ID to the file
1. Open `index.html` in any text editor (Notepad / TextEdit / VS Code)
2. Press **Ctrl+H** (Windows) or **Cmd+H** (Mac) to open Find & Replace
3. Find: `G-XXXXXXXXXX`
4. Replace with: your real ID e.g. `G-ABC123XYZ`
5. Click **Replace All** (it appears twice — both get replaced)
6. Save the file

### Step 3 — Re-upload to GitHub
1. Go to your GitHub repository
2. Click on `index.html` in the file list
3. Click the **pencil/edit icon** → scroll down → click **Upload** 
   OR simply drag the new index.html onto the repository page
4. Click **Commit changes**
5. Done — analytics activates within 24 hours

- **Devices** — phone vs laptop breakdown
